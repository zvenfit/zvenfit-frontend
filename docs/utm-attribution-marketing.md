# UTM в заявках ZvenFit — инструкция для маркетинга

Кратко: метки из рекламных ссылок сохраняются в браузере пользователя до **7 дней** и попадают в **Telegram** вместе с заявкой с формы сайта.

---

## Что настраивать в рекламе

В ссылках на сайт используйте стандартные UTM (и автоматические id кликов, если их отдаёт площадка):

| Параметр       | Пример              | Зачем                          |
|----------------|---------------------|--------------------------------|
| `utm_source`   | yandex, vk, instagram | Источник трафика             |
| `utm_medium`   | cpc, social, email  | Тип канала                     |
| `utm_campaign` | spring_pilates      | Название кампании              |
| `utm_term`     | реформер звенигород | Ключевое слово / объявление    |
| `utm_content`  | banner_a            | Креатив или вариант объявления |
| `yclid`        | (авто из Директа)   | Идентификатор клика Яндекса    |
| `gclid`        | (авто из Google Ads)| Идентификатор клика Google     |
| `fbclid`       | (авто из Meta)      | Идентификатор клика Facebook/Instagram |

**Пример ссылки:**

```
https://zvenfit.ru/?utm_source=yandex&utm_medium=cpc&utm_campaign=probnaya_mart
```

Можно вести не только на главную, но и сразу на форму:

```
https://zvenfit.ru/forma-dlya-zayavki/?utm_source=yandex&utm_medium=cpc&utm_campaign=probnaya_mart
```

---

## Как это работает для пользователя

1. Человек **кликает по рекламе** — в адресе страницы есть `?utm_...` (и при необходимости `yclid` / `gclid` / `fbclid`).
2. Сайт **запоминает** эти метки в браузере на срок до **7 дней**.
3. Пользователь **ходит по сайту** (разделы услуг, тренеры и т.д.) — в адресной строке меток уже нет, но они **не пропадают**.
4. Открывает **форму заявки**, заполняет и нажимает «Отправить».
5. В **Telegram-группу с заявками** приходит сообщение с блоком **«Маркировка»** — те же метки, что были при заходе с рекламы (если не было нового клика с другим каналом).

**Пример сообщения в Telegram:**

```
Новая заявка
Имя: Анна
Телефон: +7 (925) ...
Способ связи: Позвонить
---
Маркировка:
source: yandex
medium: cpc
campaign: probnaya_mart
yclid: ...
```

По этому блоку видно, что заявка пришла, например, с Директа и кампании `probnaya_mart`, а не «просто с сайта».

## Правила и ограничения

### Окно атрибуции — 7 дней

Если пользователь кликнул рекламу, а заявку оставил **позже чем через 7 дней**, метки могут не подтянуться (считаются устаревшими).

### Смена рекламного канала

Сценарий: сначала заход с Яндекса (`utm_source=yandex`), позже — новый клик с VK (`utm_source=vk`).

В заявке будет **VK**. Старые UTM-поля кампании от Яндекса (например, `utm_campaign`) **не смешиваются** с новым каналом.

### Заход без рекламы

Прямой ввод адреса, закладка, переход из мессенджера **без UTM** — блока «Маркировка» в Telegram **не будет**. Это ожидаемо.

### Органический трафик

Ссылка без UTM → в заявке меток не будет, даже если человек долго листал сайт перед отправкой формы.

### Редкие случаи

Если в браузере полностью заблокированы сохранение данных (cookie / local storage), метки могут не сохраниться между страницами. Тогда UTM попадут в заявку только если они **есть в URL** в момент отправки (например, реклама ведёт сразу на форму с метками в ссылке).

## Что проверить после выкладки на сайт

1. Открыть тестовую ссылку:
   ```
   https://zvenfit.ru/?utm_source=test&utm_medium=cpc&utm_campaign=check
   ```
2. Перейти на любую внутреннюю страницу (без UTM в адресе).
3. Открыть форму заявки и отправить тест.
4. В Telegram должны появиться строки:
   - `source: test`
   - `medium: cpc`
   - `campaign: check`

---

## Чего эта схема не делает

- **Не заменяет** Яндекс.Метрику, Google Analytics или GTM — там по-прежнему аналитика визитов и поведения на сайте.
- **Не отправляет** данные в CRM автоматически — только в Telegram вместе с заявкой.
- **Не даёт** 100% атрибуцию во всех браузерах и режимах приватности (как и любая клиентская схема на сайте).

---

## Рекомендации по разметке кампаний

- Всегда заполняйте минимум **`utm_source`** и **`utm_medium`**.
- Для каждой кампании задавайте уникальный **`utm_campaign`** (латиница, без пробелов — используйте `_` или `-`).
- Разные креативы различайте через **`utm_content`**.
- Для Яндекс.Директа не удаляйте **`yclid`** из финальных URL, если площадка его подставляет.

---

## Итог одной фразой

Настраиваете UTM в рекламе как обычно → клиент может спокойно изучать сайт → в Telegram видите, с какой кампании и канала пришла заявка, в течение **7 дней** после рекламного клика.
